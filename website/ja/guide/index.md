# VMark をはじめる

VMarkは、デュアル編集モード、豊富なフォーマットツール、優れたCJK（中国語/日本語/韓国語）サポートを備えたローカルファーストのMarkdownエディタです。

## クイックスタート

1. [ダウンロードページ](/ja/download)からVMarkを**ダウンロードしてインストール**する
2. **アプリを起動**してすぐに執筆を開始する
3. `Cmd/Ctrl + O` で**ファイルを開く**か、`.md` ファイルをドラッグ＆ドロップする
4. ワークスペースモードで `Cmd/Ctrl + Shift + O` で**フォルダを開く**

## インターフェースの概要

### メインエリア

- **エディタ**: ドキュメントを作成するメインの執筆エリア
- **サイドバー**: ファイルツリーナビゲーション（`Ctrl + Shift + 2` で切り替え）
- **アウトライン**: ドキュメント構造ビュー（`Ctrl + Shift + 1` で切り替え）
- **ステータスバー**: 単語数、文字数、自動保存状態（`F7` で切り替え）
- **ターミナル**: 統合シェルパネル（`` Ctrl + ` `` で切り替え）

### メニューバー

- **ファイル**: 新規、開く、保存、エクスポート操作
- **編集**: 元に戻す/やり直し、クリップボード、検索/置換、ドキュメント履歴
- **ブロック**: 見出し、リスト、引用、行操作
- **フォーマット**: テキストスタイル、リンク、テキスト変換
- **表示**: エディタモード、サイドバー、フォーカス/タイプライターモード
- **ツール**: テキストクリーンアップ、CJKフォーマット、画像管理

### 編集モード

VMarkは2つの編集モードをサポートしており、切り替えることができます：

| モード | 説明 | ショートカット |
|--------|------|--------------|
| リッチテキスト | ライブフォーマット付きのWYSIWYG編集 | デフォルト |
| ソース | シンタックスハイライト付きの生のMarkdown | `F6` |

### 表示モード

これらの表示モードで執筆に集中できます：

| モード | 説明 | ショートカット |
|--------|------|--------------|
| フォーカス | 現在の段落をハイライト | `F8` |
| タイプライター | カーソルを中央に固定 | `F9` |
| ワードラップ | 行の折り返しを切り替え | `Alt + Z` |

## 基本的なフォーマット

### テキストスタイル

| スタイル | 構文 | ショートカット |
|---------|------|--------------|
| **太字** | `**text**` | `Cmd/Ctrl + B` |
| *斜体* | `*text*` | `Cmd/Ctrl + I` |
| ~~取り消し線~~ | `~~text~~` | `Cmd/Ctrl + Shift + X` |
| `コード` | `` `code` `` | `Cmd/Ctrl + Shift + `` ` `` |

### ブロック要素

- **見出し**: `#` 記号または `Cmd/Ctrl + 1-6` を使用
- **リスト**: `-`、`*`、`1.`、またはタスクリストの場合 `- [ ]` で行を開始
- **引用**: `>` で始めるか `Alt/Option + Cmd + Q` を使用
- **コードブロック**: オプションの言語指定付きでトリプルバッククォートを使用
- **テーブル**: フォーマットメニューまたは `Cmd/Ctrl + Shift + T` を使用

## ファイルの操作

### 作成と開く

- **新規ファイル**: `Cmd/Ctrl + N`
- **ファイルを開く**: `Cmd/Ctrl + O`
- **フォルダを開く**: `Cmd/Ctrl + Shift + O`（ワークスペースモード）

### 保存

- **保存**: `Cmd/Ctrl + S`
- **名前を付けて保存**: `Cmd/Ctrl + Shift + S`
- **自動保存**: デフォルトで有効、設定で変更可能

### エクスポート

- **HTMLとしてエクスポート**: **ファイル → HTML としてエクスポート** — VMark Readerを含むインタラクティブなHTMLを生成
- **PDFとしてエクスポート**: 印刷（`Cmd/Ctrl + P`）からPDFとして保存
- **HTMLとしてコピー**: `Cmd/Ctrl + Shift + C`

エクスポートされたHTMLには、目次、設定パネルなどを備えたVMark Readerが含まれます。[詳細を見る →](/ja/guide/export)

## 設定

`Cmd/Ctrl + ,` で設定を開いてカスタマイズできます：

- **外観**: テーマ、フォント、フォントサイズ、行の高さ
- **エディタ**: 自動保存間隔、デフォルトの動作
- **ファイルと画像**: アセット管理、ドキュメントツール
- **統合**: AIプロバイダー、MCPサーバー
- **言語**: CJKフォーマットルール
- **Markdown**: エクスポートオプション、フォーマット設定
- **ショートカット**: キーボードショートカットのカスタマイズ
- **ターミナル**: ターミナルのフォントサイズと行の高さ

## AIライティング支援

VMarkにはビルトインのAI Genieが搭載されています — テキストを選択して `Mod + Y` を押すと、AIで文章を磨いたり、拡張したり、翻訳したり、変換したりできます。**設定 > 統合**で好みのプロバイダーを設定してください。

[AI Genieについて詳しく →](/ja/guide/ai-genies) | [プロバイダーの設定 →](/ja/guide/ai-providers)

## はじめてのヒント

1. **アウトラインでナビゲート**: アウトラインアイテムをクリックしてセクションにジャンプ
2. **AI Genieを試す**: テキストを選択して `Mod + Y` を押して磨く、言い換える、拡張する
3. **フォーカスモードを試す**: `F8` で現在の段落に集中しやすくなる
4. **ショートカットを学ぶ**: 効率化のために[ショートカットガイド](/ja/guide/shortcuts)をチェック
